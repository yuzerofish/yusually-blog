# Blog Starter Promo Video

Chinese Remotion promo film for `01mvp-blog-starter`.

```bash
pnpm -F @repo/promo-video dev
pnpm -F @repo/promo-video still
pnpm -F @repo/promo-video render
```

The audio is generated locally by `scripts/generate-audio.mjs`. It creates an original ambient music bed and short UI sound effects under `public/audio/`.
