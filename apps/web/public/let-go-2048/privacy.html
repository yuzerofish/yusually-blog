Privacy notes

Let Go 2048 is currently prepared as a local iPhone mini-game. The current app build does not include accounts, advertising SDKs, analytics SDKs, cloud sync, or third-party tracking.

Data collection

No personal data is intentionally collected by the current app. Gameplay state is local to the device and is not uploaded by the app.

External links

The About page and this website may link to GitHub or yusually.it.com. Those external sites have their own privacy practices.

Release status
