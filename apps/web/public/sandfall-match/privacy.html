Privacy

Sandfall Match does not require an account and does not use ads, analytics SDKs, tracking, or network data collection in the current app.

Gameplay happens on device. If an optional App Store tip jar is enabled in a future build, purchases are processed by Apple using StoreKit.

中文说明：当前应用无需账号，不接入广告、分析 SDK、追踪或网络数据收集。若未来启用可选打赏，将通过 Apple StoreKit 应用内购买处理。
