真实的水之道

我站在那些帆船和独木舟前，突然意识到：

《阿凡达：水之道》不是凭空幻想。
它的真实版本，早已在太平洋上航行了几千年。

船不是船，是身体

速度、平衡、承重和捕风，全部被做进结构里。它更像一副能在海上呼吸的骨架。

外伸浮木 像平衡翼，让船把海浪变成侧向支撑。
双体船 像并行的身体，把速度、容量和稳定性一起放大。
蟹爪帆 像捕风的翅膀，把风向翻译成航线。
植物纤维捆扎 像肌腱，不靠金属，也能让结构保持韧性。
异常尺寸 巨大不是为了夸张，而是为了让海成为道路。

当它被悬挂在展厅里，船身突然不像交通工具，而像一具来自海洋文明的完整身体。

部落不是装饰，是世界观

Pasifika / Polynesian / Māori / Tokelau

这些船不是孤立的物件。它们属于祖先知识、迁徙路线、岛屿关系，也属于今天的身份和气候危机。
