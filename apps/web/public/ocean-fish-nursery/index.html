这个小游戏是怎么做出来的

先让 AI 生成像素素材，再用 Canvas 写一个简单养成循环。

实现逻辑

  1. 确定玩法闭环：投喂 → 小鱼吃饲料 → 成长 → 金鱼收益 → 买鱼或清水。
  2. 用 imagegen 生成海底背景和透明鱼类素材。
  3. 用 HTML Canvas 绘制背景、鱼、饲料、泡泡和海草。
  4. 用 JavaScript 保存游戏状态：金币、天数、水质、鱼群、饲料。
  5. 每一帧让鱼游向目标；如果附近有饲料，就游过去吃并增加成长值。

提示 AI 的通用写法

告诉 AI 四件事：主题、玩法、技术、素材风格。越具体，越容易一次做成可玩的版本。

做一个可以直接在浏览器运行的小游戏。
主题：海洋里养小鱼。
技术：HTML + CSS + JavaScript Canvas，不依赖后端。
玩法：点击投喂，小鱼会游过去吃；吃够后长成金鱼；金鱼能赚钱；可以清洁水质和买新鱼。
界面：第一屏就能玩，要有清楚的新手提示和工具栏。
素材：使用像素风海底背景、小鱼、饲料、海草、泡泡。
