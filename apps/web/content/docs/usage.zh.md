这个网站模板就像一把瑞士军刀——不同场景用不同的"刀片"。下面按你可能遇到的场景，一个个说。

## 想写一篇文章？

**去管理后台，在浏览器里写就行。**

打开 `/admin`（你网站地址后面加 `/admin`），登录后就能：

- 像在飞书文档里一样写文章（支持 Markdown 格式）
- 随时保存草稿，不用担心丢失
- 上传封面图和配图
- 写完立即发布，或者设定一个时间自动发布
- 管理读者评论

发布后，文章会自动出现在博客页面、标签页、RSS 订阅源里。**你不需要碰任何代码。**

## 想写产品文档？

**直接编辑仓库里的 Markdown 文件。**

产品文档（就是你现在看到的这些页面）放在项目文件夹里。如果你用过飞书文档或 Notion，可以把它理解成"用纯文本写的文档，但有格式"。

- 英文文档：`文件名.md`
- 中文文档：`文件名.zh.md`

> 不熟悉 Markdown？它就像微信里的"加粗""列表"快捷键，只不过用符号代替了按钮。比如 `**加粗**` 就是加粗，`- 列表项` 就是列表。5 分钟就能学会。

## 想让 AI 帮你发文章？

**用 OpenAPI 接口，让 AI 工具自动发布。**

这个网站提供了一套标准的 API（可以理解成"机器能读懂的操作说明书"）。你可以：

1. 在管理后台的设置页，创建一个 API Token（相当于给机器一把钥匙）
2. 把这个"钥匙"给你的 AI 工具或自动化脚本
3. AI 就能帮你自动发布文章、管理内容

适合的场景：批量发布、AI 写完自动上线、对接其他工具。

## 想部署上线？

**一条命令搞定。**

当你准备好让网站对外可见时：

```sh
pnpm deploy:web
```

> 这条命令就像按下"发射按钮"——它会自动完成构建、数据库迁移、内容同步等所有步骤。如果你有技术同事帮你操作，把这条命令发给他就行。
